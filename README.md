## Getting Started

### Installation

```sh
# Lerna
yarn global add lerna

# Dependencies
yarn install
```

### Run

```sh
yarn dev
```

### Test

```sh
yarn test
```

### Build

```sh
yarn build
```

#### Start

```sh
yarn start
```
